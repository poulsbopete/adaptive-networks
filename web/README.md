# Demo UI (legacy path)

The Next.js demo app now lives at the **repository root** (`app/`, `lib/`, `package.json`).

- **Vercel:** leave Root Directory as `.` (default) — Next.js is detected from root `package.json`.
- **Local dev:** `npm install && npm run dev` from the repo root.

See the main [README](../README.md#vercel-demo-ui) for environment variables and deploy steps.
